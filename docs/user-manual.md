# 用户手册

## 创建页面
1. 访问 /admin 管理后台
2. 点「创建页面」，填标题提交
3. 自动进入编辑器

## 使用编辑器
- 左侧组件库：拖拽组件到画布
- 画布：选中组件，右侧配置属性
- 顶部工具栏：主题选择、设备预览（桌面/平板/手机）、保存、发布
- 点「另存为模板」保存当前页面为模板

## 主题管理（/admin/themes）
- 5 个预设主题，可编辑颜色变量
- 导入 JSON 主题文件
- 在编辑器中选择主题实时切换

## 媒体管理（/admin/media）
- 上传图片（单文件/批量）
- 网格列表展示，支持搜索
- 在组件配置中使用 MediaPicker 选择

## 模板管理（/admin/templates）
- 5 个预设模板
- 点「使用此模板」快速创建页面
- 编辑器点「另存为模板」

## SEO
在页面列表点「SEO」编辑 meta 标题/描述/关键词/OG 图片
静态导出的 HTML 包含这些 SEO 标签
